## Instructions to Run the Script
1. Ensure you have Python installed (version 3.6 or higher recommended).

3. Place the “titanic.csv” file within the project directory.
4. Run the script from the command line: “python analyze_data.py”.
5. The script will execute from start to finish, printing all outputs clearly labeled to the console.
